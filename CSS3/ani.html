애니메이션 속성

  1. 웹 요소에 애니메이션 속성을 추가
  2. 애니메이션을 시작해서 끝나는 동안 원하는 곳에 스타일을 바꾸어 애니메이션 정의 가능
  3. 키프레임(keyframe) : 애니메이션 중간에 스타일이 바뀌는 지점을 정의
  4. 애니메이션 속성
    • @keyframes : 애니메이션이 바뀌는 지점을 정의
      • 형식 : @keyframe 이름(키프레임명)(필수) { 0%{ /*CSS 코드 */} n%{ /*CSS 코드 */} 100%{ /*CSS 코드 */} }
      • 이름으로 애니메이션을 구분
      • 시작점 0%, 끝점 100%로 놓고 지정
      • 시작, 끝만 사용시 from, to 키워드 사용가능
    • animation-name 속성(필수)
      • 애니메이션을 지정할 키프레임명 지정
      • 애니메이션을 구분하는 역할
      • @keyframe에서 지정한 이름을 사용
      • 형식 : animation-name: 이름 | none
    • animation-duration 속성값(필수)
      • 애니메이션의 지속시간 지정
      • 설정 값은 초(s), 밀리초(ms)
    • animation-delay 속성
      • 애니메이션의 지연 시간 설정
      • 형식 : animation-delay : 지연시간
    • animation-direction 속성
      • 애니메이션의 진행 방향을 지정
      • 형식 : animation-direction: 속성값
      • normal: 애니메이션의 진행 방향을 키 프레임에 정의된 시간 순서대로 진행(to->from)
      • reverse : 애니메이션의 진행 방향을 키 프레임의 역순으로 진행(from->to)
      • alternate: 1회 이상 실행시에는 홀수번째는 normal. 짝수번째는 reverse로 진행
      • alternate-reverse: 홀수버째는 reverse, 짝수번째는 normal
      • animation-iteration-count 속성
        • 애니메이션의 반복 횟수 지정
        • 형식 : animation-iteration-count: 횟수
        • 숫자 : 반복횟수 지정
        • infinite : 무한 반복
      • animation-timing-function 속성
        • 애니메이션의 속도 지정
      • animation-play-state 속성
        • 애니메이션의 진행/정지 상태 정의
        • 형식 : animation-play-state: 속성값
        • paused : 일시 정지
        • running : 실행
      • animation 속성
        • 모든 애니메이션 속성을 지정
        • 형식 : animation: name duration timing-function delay iteration-count direction fill-mode play-state
        • 지정하지 않은 속성은 기본값 사용
        • animation-duration 값은 필수
