트랜지션

  1. 웹 요소의 스타일 속성을 시간차를 두고 조금씩 바뀌게 설정하는 것
  2. 어떤 요소의 속성 값을 다른 값으로 변경하는 것
  3. transition-property 속성
    • 트랜지션의 대상을 지정
    • 전화 효과의 새당 속성명을 값을 지정
    • 형식 : transition-property:<속성값>
    • none : 전환 효과 속성을 지정 안함
    • all : 모든 속성을 전환 효과 대상으로 지정
    • 속성 이름 : 트랜지션 효과를 적용할 속성이름, 콤마(,)로 구분
    • width, height, 배경색 등
  4. transition-duration 속성값
    • 트랜지션 진행 시간 설정
    • 어떤 요소에 전환 효과를 적용시 property, duration을 함께 사용
    • 여러개 일 경우, 콤마(,)로 구분하여 적용
  5. transition-delay 속성값
    • 전환 효과가 지연되는 시간을 결정
    • 형식 : transition-delay:<시간>
  6. transition-timing-function 속성값
    • 트랜지션의 전환 효과 속도를 지정
    • 트랜지션의 시작, 중간, 끝 지점의 속도를 지정
    • 형식 : transition-timing-function: <속성값>
    • ease : 처음에는 빨리 진행하다가 중간 지점부터 점점 느려짐
    • lenear : 처음, 마지막 속도 일정
    • ease-in : 처음에는 느리다가 점점 빨라짐
    • ease-out: 처음에는 빠르다가 점점 느려짐
    • ease-in-out : 처음에는 느리다가 점점 빨라지다가 다시 점점 느려짐
    • cubic-bezier(n,n,n,n) : 베지에 함수 이용하여 정의, n값은 0~1사이값
  7. transition 속성
    • 위의 속성값을 한꺼번에 지정 가능
    • 형식 : transition: property, duration, timing-function, delay
    • 시간 속성을 앞에 것은 duration, 뒤에 것은 delay로 간주
