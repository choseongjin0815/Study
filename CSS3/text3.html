  4. text shadow

    • 가로거리 : 테스트로부터 그림자의 가로거리, 필수 속성
    • 세로거리 : 텍스트로부터 그림자의 세로 거리, 필수
    • 번짐 정보 : 그림자가 번져 보이는 정도, 기본값 0
    • 색상 : 그림자의 색상 지정, 여러가지 지정 가능
