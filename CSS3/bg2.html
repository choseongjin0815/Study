1. bakcground-image 속성
    • 배경 이미지 파일의 경로를 지정
    • 기본적으로 이미지의 전체 사이즈가 지정
    • url함수 이용하여 지정 -이미지 태그의 경로 지정과 동일
  2. backgorund-repeat 속성
    • background-repeat:repeat | repeat-x | repeat-y | no-repeat
    • 배경 이미지의 반복 출력 속성 값을 설정
    • 영역이 이미지보다 클때 - 이미지 반복
    • 영역이 이미지보다 작을 때 - 영역 크기만큼 잘려 보임
    • repeat : 해당 영역이 가득 찰때까지 배경의 가로 세로를
    • repeat-x : 해당 영역이 가득 찰때까지 배경의 가로로 반복
    • repeat-y : 높이와 같아질 때까지 이미지를 세로로 반복
    • no-repeat : 반복없이 한번만 보여줌
