1. 원형 그라데이션
    1. radial-gradient(<모양> <크기> at 위치, <색상 중지점>, [<중지점>,,,,,])
    2. 원이나, 타원 중심부터 동심원을 그리면서 바깥방향으로 색상이 바뀌는 그라데이션
    3. 색상 시작하는 점 원 중심, 모양 지정
  2. 모양 : circle(원형), ellipse(타원형)-ellipse
  3. 위치 : 그라데이션 시작 위치
    1. 모양 크기 at 위치값
    2. 위치값 : 백분율, 키워드
    3. 수평 : left, center, right
    4. 수직 : top, center, bottom
  4. 색상 중지점
    1. 색상이 바뀌는 지점에서 노출될 색상, 구분하여 설정
    2. 색상이 변경되는 지점 설정 가능
