객체 자료형은 스택에는 번지수가 들어가고 값은 힙에 들어간다.

표준 내장 객체

  • 자바 스크립트에 기본으로 내장된 객체
  • 문자열 객체, 문자(char), 문자열(string) 자바스크립트에서는 '',""를 구분하지 않음
  • Array 객체
    • 배열과 관련된 속성, 메서드가 미리 정의되어 있음
    • 파괴적 메서드 : 기본 메서드를 사용했을 때, 원본 데이터가 변경되는 메서드
    • 비파괴적 메서드 : 기본 메서드를 사용했을 때, 원본데이커가 영향을 받지않는 메서드
    • let 변수명 : [초기화]
    • new
  • 날짜
    • 날짜와 시간을 다루는 Date 객체
    • get 메서드 : 날짜와 시간 정보를 가져오는 메서드
    • set 메서드 : 날짜와 시간 정보를 설정하는 메서드
    • let 변수명 = new Date();
  • 수학 연산
    • 수학 연산과 관련있는 속성과 메서드가 정의된 객체, Math
    • random() 메서드 : 0 ~ 1 사이의 난수를 반환하는 메서드
