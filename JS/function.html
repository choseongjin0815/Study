함수(function)

  • 어떤 목적을 가지고 작성한 코드를 모아놓은 블록문
  • 각 명령의 시작과 끝을 명확히 구분할 수 있음
  • 함수를 정의한다 : 블록문을 구성한다.
    • 블록문을 구성한다.
    • function 키워드
    • 식별자
    • 소괄호
  • 코드의 재활용이 가능
  • function 함수명(매개변수...) { 실행구문; 함수; 로직; 변수; [return 값;] }
  • 함수표현식
    • 익명 함수(많이 씀)
      • 보통 단발적으로 끝나는 애들 인라인 코드가 많음
      • let 변수명 = function (매개변수) {}
      • 변수명(매개변수,,,,,)
      • 변수명(10,20)
      • function sum(매개변수1, 매개변수2) {}, sum(10, 20)
    • 네이밍 함수
      • 변수에 할당하는 함수에 식별자(함수명)가 있을 때
      • const 변수명 = function 이름(){};
      • const a = {}
    • 즉시 실행 함수
      • 실행 위치가 굉장히 중요함
      • 페이지가 로딩될때 즉시 실행
      • (function(){실행구문}());
      • (function(매개변수,,,,,){실행구문,,,,}(인수));
    • 화살표 함수(많이 씀)
      • ES6에 정의
      • ()=>{}
      • const sum = ()=>{로직}; sum();
