• for 문

    for (초기값; 조건; 증가식) { 실행문; 조건문; 반복문; 함수; 선언문; ..... }

  • 초기값 : for 문에서 사용하는 반복 조건값을 초기화
  • 조건 : 조건이 true 동안에 for문이 반복
  • 증가식 : for문이 반복되는 동안 초기값을 증가시킴

    • for(i = 0 ;true;)

    for(;;) { }

    자바스크립트에서 파라미터 명을 지역변수든 전역변수든 동일한것 사용하지 않는다.
