자바스크립트의 자료형

  1. 숫자형
    • 숫자 형태의 데이터를 처리
    • 정수 : -3, -2, 0 , 1 , 10
    • 실수 : -0.25, 0, 3.0
    • 실수의 소수점 이하 자리수에 대하여 정밀도를 보장하지 않는다.(다를때도 있다.)
  2. 문자열 (string)
    • 문자가 한개 이상 이어져 있는 것을 문자열
    • 자바스크립트에서는 'a', "a" 둘다 문자열이고 구분안함
  3. 논리형(boolean)
    • 참(true) 또는 거짓(false) 값을 표현
    • 조건을 확인하여 맞으면 true, 틀리면 false
  4. undefined 유형
    • 자료형이 정의되지 않았을 때의 데이터 상태
    • 변수 선언만 하고 값이 할당되지 않은 상태
  5. null 유형
    • 데이터 값이 유효하지 않은 상태
    • 변수에 할당된 값이 유효하지 않다는 의미
  6. 배열(array)
    • 하나의 변수에 여러 값을 저장할 수 있는 유형
    • 자료형의 유형이 같은 값을 할당
  7. 객체
    • 배열도 객체의 일종
    • 함수와 속성을 함께 포함
    • 배열, 함수, 객체 리터럴
    • 객체 리터럴 : 중괄호({})를 사용, 키(key), 값(value) 한 쌍으로 이루어진 데이터
  8. var name;
  9. 연산자의 종류
    • +, -, *, / : 수학과 동일
    • % : 퍼센트 연산자, 모듈러 연산자 - 나머지 연산자
    • ++: 피연산자를 1증가
    • **: 제곱승 ==>a**a, a^a
